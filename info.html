Cara Login Siswa

  1. Klik tombol "Login Siswa" di halaman utama
  2. Pilih login menggunakan Email atau NIS
  3. Masukkan email atau NIS sesuai pilihan Anda
  4. Masukkan password yang sesuai
  5. Klik tombol "Masuk"
  6. Jika berhasil, Anda akan diarahkan ke dashboard siswa

Tips Penting

  • Password bersifat case-sensitive (bedakan huruf besar dan kecil)
  • Gunakan email atau NIS saat login, tidak keduanya
  • Jika lupa password, hubungi admin di halaman "Lihat Data Login Siswa"
  • Data siswa tersimpan di localStorage browser Anda
  • Hapus cookies jika ada masalah login

Jika Gagal Login

  • Pastikan email/NIS dan password sudah benar
  • Cek apakah browser JavaScript diaktifkan
  • Coba buka di tab incognito/private untuk menghindari cache
  • Buka Developer Console (F12) untuk melihat error messages
  • Jika masih bermasalah, silakan hubungi admin
